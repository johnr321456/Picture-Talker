# Contributing

Thank you for helping improve Picture AAC Talker.

Useful feedback and contributions include:

- Missing starter boards
- Missing vocabulary
- Confusing wording
- Tile size and spacing issues
- iPad, iPhone, Android, Windows, or Mac browser issues
- Accessibility issues
- Privacy or safety concerns
- Better help walkthroughs

Please avoid sharing private information about children or AAC users in issues, screenshots, exported board files, or discussions.

If you are suggesting clinical or educational changes, please describe the use case and whether the suggestion comes from lived experience, caregiver experience, educator experience, or SLP/AAC professional experience.
