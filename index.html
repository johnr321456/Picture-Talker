    Access

    Local-first Promise

    This app saves boards in this browser on this device. There is no subscription, account, cloud sync, or usage tracking built into this version.

    Privacy Guardrails

    Do not enter private medical details, addresses, real phone numbers, or identifying child information when sharing screenshots or feedback. The SMS number field is only saved in this browser's local storage on this device.

    SMS Notes

    Send SMS opens the device's Messages app with the current AAC message filled in. Receiving SMS directly inside this browser app requires a phone-number service or paired-device bridge, so this local version supports pasting a reply and speaking it aloud.

    Group Chat Notes

    The built-in Group Chat is a local demo that syncs between tabs or windows on the same device. To use separate iPads, phones, or computers in a therapy group, the app needs a backend realtime room service such as Firebase, Supabase, or a custom WebSocket server.

    Tool Reference

    Communicate

    Build a spoken message from picture tiles.

    1. Choose a page.
    2. Tap tiles in order.
    3. Check the message bar.
    4. Tap Speak.

    Find Word

    Locate vocabulary or speak a temporary phrase.

    1. Type part of a word.
    2. Tap a matching tile.
    3. Use Say Now for a phrase not on the board.

    Group Chat

    Practice turn-taking in a local room log.

    1. Enter name and room code.
    2. Build an AAC message.
    3. Send it to the room.
    4. Use Speak Last to hear a reply.

    Builder

    Prepare vocabulary before communication practice.

    1. Use Picture Word Builder for visual tap-to-add board building.
    2. Choose the Build page before adding words.
    3. Use category or search to find words.
    4. Scroll dense previews sideways if needed.
    5. Save a profile or export a backup.

    Settings

    Adjust speech and access supports.

    1. Test rate and pitch.
    2. Choose tile speech behavior.
    3. Lock editing for daily use.
    4. Use larger labels or high contrast if needed.

    Backup

    Protect custom boards and move them between browsers.

    1. Export before big edits.
    2. Store JSON files privately.
    3. Import only trusted backups.
