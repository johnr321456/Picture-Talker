Screen Guides

Message Bar

The message bar is the sentence workspace. It shows the words selected from picture tiles before they are spoken.

  1. Tap picture tiles to add words to the message bar.
  2. Use Backspace if only the last word is wrong.
  3. Use Clear when the user wants to start over.
  4. Use Speak when the message is ready.

Communicate

Communicate is the main AAC board for building messages during daily routines. The default Core page starts as a 6 x 6 board so important core words are available immediately.

  1. Choose a vocabulary page.
  2. Tap tiles in message order.
  3. Use quick phrases for common full messages.
  4. Tap Speak to say the message aloud.

Find Word

Find Word helps when a word is hard to locate or when a quick spoken phrase is needed.

  1. Type part of the word.
  2. Tap a matching tile to add it.
  3. Type a full phrase if the word is not on the board.
  4. Use Say Now to speak that phrase immediately.

SMS Tools

SMS tools help move an AAC message into another messaging app while keeping the user in control.

  1. Build the AAC message.
  2. Enter the phone number only on private devices.
  3. Use Send SMS to open the Messages app for review.
  4. Paste replies into Received Message and use Speak Received.

Group Chat

Group Chat supports turn-taking practice in a therapy activity or classroom activity.

  1. Enter a participant name and room code.
  2. Build a message on Communicate.
  3. Return to Group Chat and send the AAC message.
  4. Use Speak Last to read the newest room message.
  5. Use Speak Chat to hear the full room transcript.
  6. Use Save Chat to download a private text transcript.

Builder

Builder is for caregiver or clinician setup before the AAC user practices.

  1. Use Picture Word Builder for visual drag-and-drop board building.
  2. Start with the 6 x 6 Core board, 9 x 9 Expanded Core, another starter board, or a generated board.
  3. Add missing pages or custom picture tiles.
  4. Use Vocabulary Builder for focused target practice.
  5. Save a profile for local switching, then export a backup for moving to another device.

Picture Word Builder

Picture Word Builder is the simplest way to build a board from a large categorized word bank with 650+ vocabulary items.

  1. Choose the page you want to build.
  2. Pick a category such as Self-Advocacy, Medical and Emergency, Food, School, or Technology.
  3. Search when the category list is too broad.
  4. Tap a picture word to add it on touch devices.
  5. Drag a picture word onto the preview if using a mouse.
  6. Scroll the preview sideways when building a dense 9 x 9 board.
  7. Tap preview tiles to hear them before switching to Communicate.

Starter Boards

Starter Boards give you a ready-made vocabulary set for a routine.

  1. Open Builder.
  2. Choose the routine that matches the activity, or choose 9 x 9 Expanded Core for a denser primary board.
  3. Tap Load Starter.
  4. Practice real messages and add missing words.

Vocabulary Builder

Vocabulary Builder focuses attention on a smaller set of practice words without moving tile positions.

  1. Enter 5 to 10 target words.
  2. Turn on Show only target words if needed.
  3. Tap Apply Targets.
  4. Practice the target words on Communicate.

Settings

Settings adjust how the app sounds and how easy the board is to access.

  1. Choose a voice on Communicate.
  2. Adjust rate and pitch in Settings.
  3. Decide whether each tile speaks when tapped.
  4. Use lock editing, larger labels, and high contrast as needed.

Help

Help provides guided practice instead of only definitions.

  1. Open a guide card for a larger view.
  2. Use the full help page for step lists.
  3. Use Narrate to hear each step.
  4. Press F1 in the app for command-specific help.

Backup / Import

Backup and Import protect custom boards when testing, sharing, or moving devices.

  1. Export before large edits.
  2. Save the JSON file privately.
  3. Import only files from this app.
  4. Test the board after importing.

Profiles

Profiles save customized boards locally so you can switch between setups on the same device.

  1. Enter a profile or board name.
  2. Tap Save Profile.
  3. Watch the storage meter for space use.
  4. Choose a saved profile and tap Load Profile when needed.
  5. Delete older profiles if local storage gets high.

Privacy Limits

This prototype is local-first and not a clinical or emergency communication system.

  1. Avoid real names and phone numbers in demos.
  2. Do not include private medical details.
  3. Treat backups as private files.
  4. Use an SLP or qualified professional for clinical decisions.

Command Help

Speak, Backspace, and Clear

Use these commands to control the message bar.

  1. Speak: read the full message when the sentence is ready.
  2. Backspace: remove the last word when only the end of the message is wrong.
  3. Clear: empty the message bar when starting a new message.

Check the message bar before speaking so the AAC user hears the intended message.

Find Word and Say Now

Use this workflow when the word is hard to find or not yet on the board.

  1. Type part of the word in Find Word.
  2. Tap a matching tile to add it to the message.
  3. If no tile fits, type the needed phrase.
  4. Tap Say Now to speak the phrase without changing the board.

Add the phrase as a custom tile later if it becomes a frequent need.

Voice Selection

Use the voice list to choose the text-to-speech voice for spoken messages.

  1. Open the voice list on Communicate.
  2. Choose a voice that is clear and comfortable.
  3. Build a short message and tap Speak.
  4. Adjust rate and pitch in Settings if needed.

Different devices and browsers may offer different voices.

SMS Tools

Use SMS tools to move the AAC message into another communication app.

  1. Build the message on Communicate.
  2. Enter the contact number if using a private device.
  3. Tap Send SMS to open the device Messages app with the text filled in.
  4. Use Copy Message when SMS links are not supported.
  5. Paste a reply into Received Message and tap Speak Received.

The app cannot directly receive SMS without a phone service or paired-device bridge.

Group Chat

Use Group Chat for local turn-taking practice during a therapy or classroom activity.

  1. Enter a participant name.
  2. Enter a room code for the activity.
  3. Build an AAC message on Communicate.
  4. Return to Group Chat and tap Send AAC Message.
  5. Tap Speak Last to read the newest room message aloud.
  6. Tap Speak Chat to read the transcript aloud in order.
  7. Tap Save Chat to download the room transcript as a text file.

This prototype syncs between tabs or windows on the same device. Separate devices require a realtime backend.

Profiles and Storage

Use Profiles to keep multiple customized boards on the same device without overwriting every setup.

  1. Customize the board with starter boards, Picture Word Builder, custom tiles, settings, phrases, and target words.
  2. Type a profile name, such as Home Board, School Board, or Therapy Practice.
  3. Tap Save Profile.
  4. Use Saved Profiles to load or delete a local profile later.
  5. Watch the storage meter. If it gets high, export backups and delete older local profiles.

Profiles are saved in this browser on this device. Export Backup is still the better choice when you need a file to move to another device.

Picture Word Builder

Use Picture Word Builder when forms feel too abstract and the user or helper needs a more visual board-building workflow.

  1. Open Builder.
  2. Choose the Build page that should receive new words.
  3. Choose a category, such as Core Words, Self-Advocacy, AAC and Access, Medical and Emergency, Food and Drink, Feelings, School, Play, Body and Care, Technology, or Places and Devices.
  4. Use Search Words to narrow the vocabulary bank.
  5. Tap a picture word to add it to the selected page.
  6. On a computer, drag a picture word onto the board preview to add it.
  7. Review the board preview and continue adding words. Dense boards may scroll horizontally so tiles do not overlap.
  8. Tap a preview tile to hear how it will speak.
  9. Save a Profile after the board feels useful.

The tap-to-add workflow is included because drag-and-drop can be awkward on iPad and iPhone.

Board Generator

Use Board Generator to make a first draft board from a plain-language situation description.

  1. Enter a board name.
  2. Describe the user, routines, and settings.
  3. Choose columns and tile size.
  4. Tap Generate Board.
  5. Review the result and add missing vocabulary manually.

This uses local templates and keyword matching. It does not call an online AI service.

Add Picture Tile

Use Add Picture Tile when an important word, phrase, person, place, or routine is missing.

  1. Enter the visible label in Word.
  2. Enter the spoken text in Speak as if it should be different.
  3. Choose a picture word or symbol hint.
  4. Choose the tile type for color/category.
  5. Choose the page where the user will look for it.
  6. Tap Add Tile and test it on Communicate.
Starter Boards

Use Starter Boards to begin from a prepared routine instead of building from an empty board.

  1. Export a backup if you want to keep the current board.
  2. Choose a starter such as 6 x 6 Starter Core, 9 x 9 Expanded Core, First Words, Mealtime, Play, School, Feelings, Bathroom, Sick/Pain, or Devices.
  3. Tap Load Starter.
  4. Practice several real messages.
  5. Add missing tiles for the user and setting.
Vocabulary Builder

Use Vocabulary Builder to reduce distraction while keeping motor patterns stable.

  1. Enter a small target list for the session.
  2. Turn on Show only target words if the board is visually overwhelming.
  3. Tap Apply Targets.
  4. Practice the target words on Communicate.
  5. Turn off the target mask when the user is ready for more vocabulary.
Edit Mode

Use Edit Mode only when a helper needs to revise existing tiles.

  1. Unlock editing in Settings if needed.
  2. Turn Edit Mode on in Builder.
  3. Tap the tile that needs changes.
  4. Update the label, spoken text, or picture.
  5. Turn Edit Mode off when finished.

Keep editing locked during normal communication to avoid accidental changes.

Print, Export, and Import

Use these tools when preparing for therapy, school, sharing, or device changes.

  1. Use Print Board to create a paper support copy.
  2. Use Export Backup before major edits or before testing on another device.
  3. Save the JSON backup somewhere private.
  4. Use Import Backup to restore the saved board.
  5. Test the board after importing.

Backups can contain private custom vocabulary. Treat them as private files.

Speech and Access Settings

Use Settings to make the app easier to hear, see, and safely use.

  1. Set speech rate and pitch using a real message as the test.
  2. Turn Speak each tile on for immediate feedback or off for sentence-only speech.
  3. Keep Lock editing on during daily communication.
  4. Use Larger Labels or High Contrast if the board is hard to scan.
Privacy and Prototype Limits

Use this prototype for exploration and feedback, not as medical advice or emergency communication.

  1. Do not put private medical details in shared screenshots.
  2. Do not share real phone numbers, addresses, school names, or identifying child information.
  3. Remember that local browser storage is not the same as a secure medical record system.
  4. Work with a qualified SLP or clinical professional for communication planning.
