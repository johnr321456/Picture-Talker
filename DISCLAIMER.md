# Prototype Disclaimer

Picture AAC Talker is a prototype shared for feedback.

It is not:

- Medical advice
- Clinical guidance
- A prescribed AAC system
- Emergency communication
- A replacement for a speech-language evaluation

Families, educators, and caregivers should work with qualified speech-language professionals when making AAC decisions.

The app may contain bugs, missing vocabulary, incomplete accessibility support, or device-specific behavior. Test carefully before relying on it in any real communication setting.
